–§—É–Ω–∫—Ü–∏—è `all()` ‚Äî —ç—Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è Python, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, **–≤—Å–µ** —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–º –æ–±—ä–µ–∫—Ç–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–ø–∏—Å–∫–µ, –∫–æ—Ä—Ç–µ–∂–µ, –º–Ω–æ–∂–µ—Å—Ç–≤–µ) —è–≤–ª—è—é—Ç—Å—è –∏—Å—Ç–∏–Ω–Ω—ã–º–∏ (`True`).

## üìå –°–∏–Ω—Ç–∞–∫—Å–∏—Å
```python

```
- **`iterable`** ‚Äî –ª—é–±–æ–π –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç (—Å–ø–∏—Å–æ–∫, –∫–æ—Ä—Ç–µ–∂, —Å—Ç—Ä–æ–∫–∞, –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä –∏ —Ç. –¥.).
    
- **–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç**:
    
    - `True` ‚Äî –µ—Å–ª–∏ **–≤—Å–µ** —ç–ª–µ–º–µ–Ω—Ç—ã `iterable` —Å—á–∏—Ç–∞—é—Ç—Å—è –∏—Å—Ç–∏–Ω–Ω—ã–º–∏ (`True`).
        
    - `False` ‚Äî –µ—Å–ª–∏ **—Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω** —ç–ª–µ–º–µ–Ω—Ç –ª–æ–∂–Ω—ã–π (`False`).
        
    - `True` ‚Äî –µ—Å–ª–∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç **–ø—É—Å—Ç**.
        

## üîç –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç `all()`?

–§—É–Ω–∫—Ü–∏—è –ø—Ä–æ–≤–µ—Ä—è–µ—Ç –∫–∞–∂–¥—ã–π —ç–ª–µ–º–µ–Ω—Ç –≤ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏:

- –ï—Å–ª–∏ **–≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã** –ø—Ä–∏–≤–æ–¥—è—Ç—Å—è –∫ `True` (–Ω–µ–ø—É—Å—Ç—ã–µ —Å—Ç—Ä–æ–∫–∏, –Ω–µ–Ω—É–ª–µ–≤—ã–µ —á–∏—Å–ª–∞, –Ω–µ–ø—É—Å—Ç—ã–µ –∫–æ–ª–ª–µ–∫—Ü–∏–∏, `True`), —Ç–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True`.
    
- –ï—Å–ª–∏ **—Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç** ‚Äî `False` (–ø—É—Å—Ç–∞—è —Å—Ç—Ä–æ–∫–∞ `""`, `0`, `None`, `False`, –ø—É—Å—Ç–æ–π —Å–ø–∏—Å–æ–∫ `[]` –∏ —Ç. –¥.), —Ç–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `False`.
    
- –ï—Å–ª–∏ **–ø–µ—Ä–µ–¥–∞–Ω –ø—É—Å—Ç–æ–π –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç** (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ø—É—Å—Ç–æ–π —Å–ø–∏—Å–æ–∫ `[]`), –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True`.
    

### üîπ –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è `all()`
```python
def all(iterable):
    for element in iterable:
        if not element:
            return False
    return True
```

## üîÑ –†–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É `all()` –∏ `any()`
