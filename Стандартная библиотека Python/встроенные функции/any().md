–§—É–Ω–∫—Ü–∏—è `any()` ‚Äî —ç—Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è Python, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, –µ—Å—Ç—å –ª–∏ **—Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω** –∏—Å—Ç–∏–Ω–Ω—ã–π (`True`) —ç–ª–µ–º–µ–Ω—Ç –≤ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–º –æ–±—ä–µ–∫—Ç–µ. –≠—Ç–æ "–ª–æ–≥–∏—á–µ—Å–∫–æ–µ –ò–õ–ò" –¥–ª—è –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏.

## üìå –°–∏–Ω—Ç–∞–∫—Å–∏—Å

- **`iterable`** ‚Äî –ª—é–±–æ–π –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç (—Å–ø–∏—Å–æ–∫, –∫–æ—Ä—Ç–µ–∂, –º–Ω–æ–∂–µ—Å—Ç–≤–æ, –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä –∏ —Ç. –¥.)
    
- **–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç**:
    
    - `True` ‚Äî –µ—Å–ª–∏ **—Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω** —ç–ª–µ–º–µ–Ω—Ç `iterable` —Å—á–∏—Ç–∞–µ—Ç—Å—è –∏—Å—Ç–∏–Ω–Ω—ã–º (`True`)
        
    - `False` ‚Äî –µ—Å–ª–∏ **–≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã** –ª–æ–∂–Ω—ã–µ (`False`) –∏–ª–∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç **–ø—É—Å—Ç**
        

## üîç –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç `any()`?

–§—É–Ω–∫—Ü–∏—è –ø—Ä–æ–≤–µ—Ä—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –¥–æ –ø–µ—Ä–≤–æ–≥–æ `True` (–ª–µ–Ω–∏–≤–æ–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ):

1. –ï—Å–ª–∏ **—Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç** ‚Äî `True`, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True` (–æ—Å—Ç–∞–ª—å–Ω—ã–µ –Ω–µ –ø—Ä–æ–≤–µ—Ä—è—é—Ç—Å—è)
    
2. –ï—Å–ª–∏ **–≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã** ‚Äî `False`, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `False`
    
3. –î–ª—è **–ø—É—Å—Ç–æ–≥–æ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞** –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `False`
    

### üîπ –≠–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è `any()`

```python
def any(iterable):
    for element in iterable:
        if element:
            return True
    return False
```
